Privacy Policy

This service is used to execute software development automation tasks through a custom MCP (Model Context Protocol) server.

No personal user data is stored. Requests are processed temporarily to interact with external services such as GitHub.

The system may send code, repository names, and related technical data to GitHub APIs for the purpose of software automation.

No data is shared with third parties beyond the required external services (e.g., GitHub).

If you have any questions, please contact the developer.
